# Github Bug Detection System - Frontend

React + Vite frontend for Github Bug Detection System.

## Setup

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

Frontend will be available at `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Features

- Interactive JSON input for repository data
- Sample data loader
- Real-time bug risk prediction
- Color-coded risk scores
- Responsive design
